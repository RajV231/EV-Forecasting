## ğŸ§  Forecast Modeling

The forecast uses a compounded annual growth calculation:

```
Future EVs = Current EVs Ã— (1 + growth_rate)^years
```

Assumptions based on market growth trends and national EV policy trajectories.

Charging demand assumes:

```
1 charger per ~400 EVs
```

This ratio can be tuned for different deployment strategies.
